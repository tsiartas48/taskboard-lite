# TaskBoard Lite

Προσωπικός Διαχειριστής Tasks με Kanban Board — Ατομική Εργασία

## Τεχνολογίες
- **Backend:** Python, FastAPI, SQLAlchemy, SQLite
- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Auth:** JWT tokens
- **Cache:** In-memory cache

## Εκκίνηση

### Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn main:app --reload

### Frontend
Άνοιξε το frontend/index.html με Live Server στο VS Code.

## API Documentation
Μετά την εκκίνηση: http://127.0.0.1:8000/docs
